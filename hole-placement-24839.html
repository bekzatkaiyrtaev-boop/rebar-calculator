Область применения и пояснения

Стандарт устанавливает расположение отверстий для заклёпок и болтов в поперечном сечении горячекатаных двутавров, швеллеров и уголков, применяемых в стальных строительных конструкциях. Расположение отверстий для высокопрочных болтов предусмотрено только для уголков.

Для уголков возможно однорядное (один ряд отверстий по полке, актуально для узких полок b=45…125 мм) и двухрядное расположение (два ряда, «рядовое» или «шахматное», для широких полок b=140…250 мм). Для широкополочных уголков указаны отдельные значения диаметра отверстия для обычных болтов (классы прочности 4.6–8.8) и заклёпок, и отдельно — для высокопрочных болтов (с учётом габарита головки гаечного ключа не более 75 мм).

Отступления от табличных размеров допускаются при стыках уголков с накладками того же размера, при образовании отверстий по кондукторам и в других обоснованных случаях — с обязательной строжкой обушка накладки для плотного прилегания (раздел 4 ГОСТ).

Предельные отклонения диаметров и расположения отверстий устанавливаются по ГОСТ 14140 и нормам на конкретный вид конструкций (раздел 5 ГОСТ) — в этот справочник не включены.

В таблице 3 исходника обнаружена нестыковка (значение ширины полки «80» между 160 и 200 мм нарушает монотонный ряд размеров широкополочных уголков 140→160→180→200→220→250 по ГОСТ 8509) — интерпретировано как опечатка «180» и исправлено; также диапазон толщины «0–12» истолкован как «10–12» мм (толщина полки не может начинаться от 0). При ответственном проектировании рекомендуется сверить эти значения с первоисточником.
